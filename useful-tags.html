blockquote

The <blockquote> element is used for extended quotations. It should include the cite attribute in order to show or link to the source of the quote:

Some common inline tags

Remember, inline tags can flow next to each other and don't cause a line break before and after the element. We have seen this with links and images.

cite

Defines the title of a work (e.g. book, a song, a movie...)

abbr

Defines an abbreviation:

WHO
